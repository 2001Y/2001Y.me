```css
input::placeholder {
  line-height: 1;
  overflow-y: visible;
}
```

`overflow: visible;`では横軸が長すぎた場合にはみ出してしまうので、事前に長さを調節しておくべきだけど、一応`overflow-y: visible;`を指定している。
