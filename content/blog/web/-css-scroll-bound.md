`Reload`するとわかりますが、アクセス時に左側にスライドしたようなアニメーションで、`margin-left`をマイナスに振ってます。

```css
.target {
  animation: bound .5s forwards;
}
@keyframes bound {
  0% {
    margin-left: 0;
  }
  50% {
    margin-left: -100px; //左側に移動させる
  }
  100% {
    margin-left: 0; //元に戻す
  }
}
```
